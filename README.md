# UK Road Accident Analysis

## Overview

This project presents a comprehensive analysis of road accidents in the United Kingdom, utilizing data from 2021 and 2022. The primary objective of this analysis is to identify key trends, contributing factors, and patterns in road accidents, providing insights that can inform road safety initiatives and policy-making. The analysis covers various aspects, including accident severity, environmental conditions, vehicle types involved, and geographical distribution.

## Data Source

The analysis is based on publicly available road accident data for the United Kingdom, covering the years 2021 and 2022. Specific details regarding the data source can be found in the original Power BI report.

## Tools Used

This analysis and its accompanying visualizations were developed using:

- **Power BI Desktop**: For data modeling, analysis, and interactive dashboard creation.

## Key Findings and Analysis Highlights

The analysis revealed several significant insights into UK road accidents:

### Contributing Factors

- **Location**: A significant portion of accidents occurred in **Urban** areas (64.46%), with **Daylight** conditions accounting for 73.8% of incidents.

- **Vehicle Type**: **Cars** were involved in the vast majority of accidents (245,337), followed by Vans (24,729) and Bikes (25,132).

- **Weather Conditions**: The majority of accidents occurred in **Fine** weather conditions, followed by Raining conditions.

- **Road Type**: Single carriageways accounted for the highest number of accidents.
